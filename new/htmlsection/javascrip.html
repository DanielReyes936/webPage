What is JavaScript?

JavaScript is a scripting language used for web applications embedded in HTML pages. In JavaScript, the code is interpreted instead of compiled.

Advantages of JavaScript

JavaScript is made to be simple and easily comprehendable to be picked up by the developers and users. Being able to create dynamic content with little to no cost is also an advantage of JavaScript, since it can also be integrated with other programming languages. JavaScript is also advantageous to use since it cuts down the time necessary for compilation, since it is an interpreted language.

Disadvantages of JavaScript

Although JavaScript is an advantageous programming language that can easily be picked up by beginner developers, it is also important to note that there are several disadvantages to using JavaScript. When using JavaScript, it will be difficult to debug, and it will also be difficult and tedious to figure out the issue since the browser doesn't display the errors in the code like the console for other programming languages. Because of this, a single error in the code can make the whole page stop rendering on the browser.

Lastly, using JavaScript doesn't guarantee security since the code is easily seen by users on the browser. Given its simple nature, it also makes it simple for anyone to alter or insert codes that will result in the security being compromised.

Common Keywords in JavaScript

The var Keyword is used in JavaScript to declare variables that are function-scoped. Meanwhile, the similar let Keyword, is used for creating variables at block-level

Example for var:
for(var x = 0; x < 3; x++) { //code }; console.log(x);

Example for let:
for(let x = 0; x < 3; x++) { //code }; console.log(x);
