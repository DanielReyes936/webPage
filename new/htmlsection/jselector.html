JavaScript Selectors

JavaScript Selectors are used to select HTML elements using their id name, class name, tag name, type, or even attributes to make it easier for the developer to manipulate the HTML element or even alter its CSS.

Advantages of JavaScript Selectors

Using JavaScript Selectors are beneficial sicne they are reusable and can be utilized multiple times. It also makes it easier for the developer to pinpoint the elements> that they want to apply changes to. This makes using JavaScript easier since there are several ways in which selectors can be used to specify the elements that the developer wants to alter.

Disadvantages of JavaScript Selectors

For JavaScript Selectors, the pros definitely outweigh the cons. There is a lot of things that can be done with JS Selectors, and the only major disadvantage it has is there is no clear way of debugging or figuring out where the error might be if a typo has occurred while typing out the id or tag name of the element that is selected.

DOM Selectors

The most common selectors that is used in JavaScript are the Document Object Model or DOM Selectors. These selectors are self-explanatory, and select elements within a document with the use of JavaScript. There are 5 ways in which you can select the elements you want to alter in a DOM through selectors. Namely, these five ways are:

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

document.getElementsByTagName()

Using this method, we can select the elements that match a specific tag name. Tag names can be the type of element, such as 'h1', 'li', 'p', etc. Using this selects all the elements of that tag name, allowing changes to be made to all the elements that fall under this umbrella.

document.getElementsByClassName()

Similar to the previous selector, this method selects more specific elements, considering the class name of the element rather than the more general tag name. This allows the developer to apply different changes to elements of the same tag name, by specifying different class names for these elements.

document.getElementbyId()

Another selector that allows for a more specific return of elements is this selector, which utilizes the id name of the element rather than the general tag name, or its class name. Once again, this adds a new level of specificity to JS Selectors that developers can use when changes are to be applied only to specific elements in the document.

Another unique thing about this selector, is that it only returns the first matched element in the document, ignoring the following elements that may be using the same ID.

document.querySelector()

When the querySelector() method it used, it returns only the first element in the document that matches the selector used by the developer. Unlike the three previous selector methods, this one can utilize the tag, class, or id name of an element.

Writing the tag names require no other characters. Meanwhile, adding a '.' before the selector specified will let the selector know that you are asking it to search through the class names of the elements. Similarly, using a '#' before the selector specified will make it search through the id names.

document.querySelectorAll()

Unlike the previous selector that only returns the first element that matches, this selector method will return all the elements matching the specifications provided by the developer. The same logic applies, where tag names require no special characters, class names require a '.' beforehand, and id names require a '#'.

  • document.querySelector('h1') -> returns first element that matches tag names
  • document.querySelectorAll('.heading') -> returns first element that matches class names
  • document.querySelector('#item') -> returns first element that matches id names
  • document.querySelectorAll('h1') -> returns all that matches tag names
  • document.querySelectorAll('.heading') -> returns all that matches class names
  • document.querySelectorAll('#item') -> returns all that matches id names

Selectors in JavaScript

The document.getElementsByTagName returns an array of elements of your choice. Using selectors, you can use this to simultaneously apply changes to elements of a certain tag name.

Example for 'getElementsByTagName':
Here, the h3 text will shift to the right side of the page.

var text = document.getElementsByTagName("h5");
for(let i = 0; i < text.length; i++) {text[i].style.textAlign="right";}
